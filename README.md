# NovaTech Solutions

A modern static business website developed as part of **Project 1 – Static Webpage Design** for the Frontend Development Internship Program.

## Project Overview

NovaTech Solutions is a technology-focused landing page designed using HTML and CSS. The website presents a fictional technology company and includes sections such as Home, Services, About Us, Statistics, and Contact.

The objective of this project was to create a clean, well-structured static webpage using semantic HTML and CSS styling techniques.

## Features

- Semantic HTML structure
- Responsive layout
- Navigation bar
- Hero section
- Services section
- About Us section
- Statistics section
- Contact section
- Modern UI design
- CSS Grid and Flexbox layouts
- Mobile-friendly design

## Technologies Used
